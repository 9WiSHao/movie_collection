# movie_collection

实训作业

前端实现: vue3 + typescript

## 运行

安装依赖

```bash

pnpm install

```

启动服务

```bash

pnpm run dev

```

## 其他

-   数据库见根目录 film_collection_management_system.sql 文件
-   电影数据来源于豆瓣top250，自行爬取，见 https://github.com/9WiSHao/doubanTop250_puppeteer
-   后端见 https://github.com/chensuperman/movie-collection_b
